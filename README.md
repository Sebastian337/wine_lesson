# 1. Описание проекта "Новое русское вино"

Этот скрипт автоматически генерирует сайт с актуальным ассортиментом вин, считывая данные из Excel-файла.

## 2. Требования к окружению

- ОС: Linux, macOS, Windows
- Python: версия 3.8 или выше
- Доступ к интернету

## 3. Как установить

### Установите зависимости:
```bash
pip install -r requirements.txt
```
## 4. Подготовка данных
Для работы программы подготовьте Excel-файл с названием wine.xlsx в корневой папке проекта. Таблица должна содержать следующие колонки:

| Категория | Название | Сорт | Цена | Картинка |
| :--- | :--- | :--- | :--- | :--- |
| Белые вина | Белая леди | Дамский пальчик | 399 | belaya_ledi.png |
| Белые вина | Ркацители | Ркацители | 499 | rkaciteli.png |
| Белые вина | Кокур | Кокур | 450 | kokur.png |
| Напитки | Коньяк классический | | 350 | konyak_klassicheskyi.png |
| Напитки | Чача | | 299 | chacha.png |
| Напитки | Коньяк кизиловый | | 350 | konyak_kizilovyi.png |
| Красные вина | Черный лекарь | Качич | 399 | chernyi_lekar.png |
| Красные вина | Хванчкара | Александраули | 550 | hvanchkara.png |
| Красные вина | Киндзмараули | Саперави | 550 | kindzmarauli.png |

## 5. Запуск
Запсук из директории:
```bash
python main.py
```
По умолчанию программа ищет файл `wine.xlsx` в корневой папке. Вы можете указать другой путь к файлу данных с помощью аргумента `--filepath`:

```bash
python main.py --filepath="путь/к/вашему/файлу.xlsx"
```

- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

Если всё настроено верно, вы увидите страницу с категориями вин и актуальными товарами из вашей таблицы.
