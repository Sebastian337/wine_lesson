# 1. Описание проекта "Новое русское вино"

Этот скрипт автоматически генерирует сайт с актуальным ассортиментом вин, считывая данные из Excel-файла.

## 2. Требования к окружению

- ОС: Linux, macOS, Windows
- Python: версия 3.8 или выше
- Доступ к интернету

## 3. Как установить

### Установите зависимости:
```bash
pip install -r requirements.txt
```

## 4. Запуск

```bash
python main.py
```
- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

Если всё настроено верно, вы увидите страницу с категориями вин и актуальными товарами из вашей таблицы.
